# 货物归类游戏

一款竖屏手机适配的2D货物归类游戏！

## 🎮 游戏玩法

1. **开始游戏**：点击"开始游戏"按钮
2. **滑动投掷**：用手指按住货物，向反方向滑动
3. **瞄准货架**：根据箭头调整方向和力度
4. **分类投掷**：
   - 红色货物 → 红色货架
   - 蓝色货物 → 蓝色货架
   - 黄色货物 → 黄色货架
   - 绿色货物 → 绿色货架
   - 炸弹 → 中间炸弹货架
5. **注意血条**：扔错或漏掉货物会扣血，血条归零游戏结束

## 🎨 美术资源说明

游戏已预留图片资源替换接口：

- `assets/background.png` - 背景图
- `assets/conveyor.png` - 传送带图
- `assets/items/` - 5种货物图片
- `assets/shelves/` - 5种货架图片

如果图片不存在，会自动使用色块作为备用方案。

## 🎯 得分机制

| 货架层 | 得分 |
|--------|------|
| 第4层（最下方） | 10分 |
| 第3层 | 20分 |
| 第2层 | 30分 |
| 炸弹货架 | 0分（不加分也不罚分） |

## 💡 小技巧

- 追求高分：挑战第2层货架
- 求稳策略：先从第4层开始练习
- 利用反弹：墙壁反弹可以让你瞄准更远的目标
- 控制节奏：不要急，慢慢来，准确率更重要
